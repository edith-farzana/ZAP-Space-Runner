# ZAP Space Runner by Team ZAP

A fast pixel-style space runner game built with HTML, CSS, and JavaScript.

Players choose a rocket, dodge a storm of space obstacles, speed up with the right arrow key, protect their three hearts, and try to beat the all-time high score.

## Play Style

- Character select screen with three rocket choices
- Endless flying gameplay with score tracking
- Speed boost using the right arrow key
- Three lives shown as hearts
- High score saved in browser storage
- Many pixel-inspired hazards including UFOs, portals, planets, asteroids, gems, shields, hourglasses, alien eyes, and enemy ships
- Background color changes every 200 points to confuse and challenge the player
- Encouragement box during gameplay

## Controls

- `Up Arrow` or `W`: move up
- `Down Arrow` or `S`: move down
- `Right Arrow`: speed boost

## Run Locally

### Option 1: Open directly

Open [index.html](./index.html) in your browser.

### Option 2: Run the local server

```bash
node server.js
```

Then open:

```text
http://localhost:3002
```

## Project Files

- [index.html](./index.html): game layout and screens
- [styles.css](./styles.css): pixel UI styling and responsive layout
- [script.js](./script.js): game logic, sprites, obstacles, score, and effects
- [server.js](./server.js): small local Node.js server
